MiniscopeV4BNO055Device#

Important

To use Miniscopes with ONIX hardware, you must first use the ONIContext configuration GUI to set the headstage port to “Passthrough Mode”

A Bonsai source that wraps a DS90UB9X Raw Device device and configures it to allow control over UCLA Miniscope V4’s onboard BNO055 inertial measurement unit.

Inputs:

None

Outputs:

A single MiniscopeV4BNO055DataFrame that is polled by software and contains the data-fields of the Device To Host Data Frame specified on the BNO055 device datasheet, with the exception of clock and frame counter information, since data is acquired via software polling.

UCLAMiniscopeV4.bonsai

Configuration#

This device has no configuration.
