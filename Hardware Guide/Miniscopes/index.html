Miniscopes#

ONIX is compatible with the following head-mounted miniscopes for freely-moving calcium imaging:

UCLA Miniscope V3#

Design Repository:

daharoni/Miniscope_CMOS_Imaging_Sensor_PCB

Documentation:

http://miniscope.org/index.php/Guides_and_Tutorials

Compatibility:

PCIe Host, Breakout Board

Bonsai.ONIX Node:

MiniscopeV3Device

UCLA Miniscope V4#

Design Repository:

Aharoni-Lab/Miniscope-v4

Documentation:

Aharoni-Lab/Miniscope-v4

Compatibility:

PCIe Host, Breakout Board

Bonsai.ONIX Nodes:

MiniscopeV4Device, MiniscopeV4BNO055Device
